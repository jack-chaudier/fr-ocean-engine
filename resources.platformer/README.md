# Platformer Example

Side-scrolling platformer demo featuring movement, jumping, collectibles, and a simple enemy.

## Run
```bash
python3 scripts/run_game.py platformer
```

Or:
```bash
./build/bin/game_engine --resources resources.platformer/
```

## Controls
- Move: WASD or Arrow Keys
- Jump: Space
- Restart: R
- Pause (time scale): P
- Quit: ESC

## Goals
- Collect coins
- Stomp the enemy from above
